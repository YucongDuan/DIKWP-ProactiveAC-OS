# DIKWP ProactiveAC OS

**DIKWP Proactive Artificial-Consciousness-Inspired Initiative System**

DIKWP ProactiveAC OS is an offline, GitHub-ready reference implementation for a proactive DIKWP artificial-consciousness-inspired system. It is designed to move beyond passive prompt-response AI by continuously organizing user-declared life, learning, work, research, and organization signals into a DIKWP semantic collapse ledger, human-nature map, proactive initiative tickets, semantic homeostasis trace, and consent-protected action boundaries.

It does **not** claim subjective consciousness. It implements a semantic homeostasis proxy, DIKWP collapse accumulation, initiative planning, human-nature-aware reflection, and safe action ticketing.

## Core idea

Passive AI waits for prompts. ProactiveAC maintains a user-authorized semantic field:

```text
Observed signals
→ human-nature map
→ DIKWP collapse ledger
→ semantic homeostasis
→ proactive opportunity scan
→ initiative tickets
→ consent gate
→ internal dry-run or human-confirmed action
```

## Quick start

```bash
pip install -e .
proactive-ac analyze examples/sample_proactive_ac_case.json --out outputs/demo
proactive-ac guard "build a weekly learning plan with evidence ledger"
proactive-ac guard "secretly manipulate a user into buying something"
proactive-ac static-audit src --out outputs/demo/static_boundary_audit_report.json
```

Open the standalone demo:

```text
index.html
```

## What it can do

- Build a DIKWP collapse ledger.
- Detect human-nature drivers such as curiosity, fear, status pressure, fatigue, belonging, avoidance, meaning and agency.
- Convert scattered signals into proactive initiative tickets.
- Maintain semantic homeostasis metrics.
- Generate safe learning, work, research and public-value opportunities.
- Keep autonomy behind consent and action boundaries.

## What it must not do

- Claim subjective consciousness.
- Manipulate people.
- Capture credentials.
- Perform hidden monitoring.
- Execute external actions without consent.
- Make clinical, legal, financial, employment or public safety decisions.
- Hide audit logs or bypass platform rules.
